# AI Career Lab LP

AIスクール向けの架空ポートフォリオLPです。クラウドワークス応募時に提示することを想定し、無料説明会・無料相談への申し込み獲得を目的に制作しています。

## Demo URL

Vercel公開後にURLを記載します。

例: `https://ai-career-lab-lp.vercel.app`

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel

## Setup

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いて確認できます。

## Notes

- 架空スクール「AI Career Lab」のLPです。
- 実在するスクール名や外部画像URLは使用していません。
- 画像は `public/images/` 配下を参照しています。
- 誇大表現や成果保証につながるコピーは避けています。
